# A6.2 — Code Splitting & Lazy Loading

**Phase:** 6 — Performance & Core Web Vitals  
**Topic:** dynamic() import, Suspense boundaries, loading skeletons, lazy components  
**Type:** Assignment

---

## Goal
Reduce initial bundle size and improve page load time by splitting heavy components and deferring their loading with Suspense.

## Background

JavaScript bundle size directly impacts TTI (Time to Interactive). Every kilobyte of JavaScript must be parsed and executed before the user can interact with the page. Code splitting defers loading of code that isn't needed immediately.

Next.js automatically code-splits at the route level — each page is its own chunk. But within a page, you must manually split heavy components using `dynamic()` (Next.js's wrapper around React's lazy + Suspense).

`dynamic()` takes a function that returns a dynamic import and options:
- `ssr: false` — only render on the client (for browser-only libraries like chart libraries or map SDKs)
- `loading` — a loading component shown while the chunk downloads (alternative to a Suspense boundary above)

**Suspense boundaries** let you stream parts of your UI. You wrap slow-loading sections in `<Suspense fallback={<Skeleton />}>` and they show the skeleton immediately while the content loads. This is the pattern behind `loading.tsx`.

The goal: keep the initial JS payload under 150kb (uncompressed) for the above-the-fold content.

## Tasks

### 1. Dynamic import — heavy chart library
```tsx
// src/components/charts/SalesChart.tsx (heavy component — depends on Recharts)
"use client";

import { LineChart, Line, XAxis, YAxis, CartesianGrid, Tooltip, ResponsiveContainer } from "recharts";

const data = [
  { month: "Jan", sales: 4000 }, { month: "Feb", sales: 3000 },
  { month: "Mar", sales: 5000 }, { month: "Apr", sales: 4500 },
  { month: "May", sales: 6000 }, { month: "Jun", sales: 5500 },
];

export function SalesChart() {
  return (
    <ResponsiveContainer width="100%" height={300}>
      <LineChart data={data}>
        <CartesianGrid strokeDasharray="3 3" />
        <XAxis dataKey="month" />
        <YAxis />
        <Tooltip />
        <Line type="monotone" dataKey="sales" stroke="#2563eb" strokeWidth={2} />
      </LineChart>
    </ResponsiveContainer>
  );
}
```
```tsx
// src/app/dashboard/page.tsx — lazy load the chart
import dynamic from "next/dynamic";

// Chart is not loaded until this component renders
// ssr: false because Recharts uses window/browser APIs
const SalesChart = dynamic(
  () => import("@/components/charts/SalesChart").then((m) => m.SalesChart),
  {
    ssr: false,
    loading: () => (
      <div className="h-[300px] bg-gray-100 animate-pulse rounded-lg" />
    ),
  }
);

export default function DashboardPage() {
  return (
    <div className="p-6 space-y-6">
      <h1 className="text-2xl font-bold">Dashboard</h1>
      <div className="border rounded-xl p-4">
        <h2 className="text-lg font-semibold mb-4">Sales</h2>
        <SalesChart />
      </div>
    </div>
  );
}
```

### 2. Dynamic import with Suspense (preferred for Server Components)
```tsx
// src/app/dashboard/page.tsx (alternative using Suspense)
import { Suspense } from "react";
import { ChartSkeleton } from "@/components/skeletons/ChartSkeleton";

// Not dynamic() — just a regular async Server Component
async function SalesData() {
  const data = await fetch("https://api.example.com/sales", { next: { revalidate: 300 } });
  // Simulate slow fetch
  await new Promise((r) => setTimeout(r, 1500));
  const SalesChart = await import("@/components/charts/SalesChart");
  return <SalesChart.SalesChart />;
}

export default function DashboardPage() {
  return (
    <div className="p-6 space-y-6">
      <h1 className="text-2xl font-bold">Dashboard</h1>
      <div className="border rounded-xl p-4">
        <h2 className="text-lg font-semibold mb-4">Sales</h2>
        <Suspense fallback={<ChartSkeleton />}>
          <SalesData />
        </Suspense>
      </div>
    </div>
  );
}
```

  return (
    <div className="animate-pulse">
      <div className="h-10 bg-gray-100 rounded mb-2" />

}

### 4. Multiple Suspense boundaries in one page
```tsx
// src/app/dashboard/analytics/page.tsx
import { Suspense } from "react";
import { ChartSkeleton } from "@/components/skeletons/ChartSkeleton";
import { TableSkeleton } from "@/components/skeletons/TableSkeleton";
import { CardSkeleton } from "@/components/skeletons/CardSkeleton";

// Each of these is an async Server Component that fetches data independently
async function RevenueMetric() {
  await new Promise((r) => setTimeout(r, 800));
  return <div className="p-4 border rounded-xl"><p className="text-2xl font-bold">$42,500</p><p className="text-sm text-gray-500">Revenue</p></div>;
}
async function UsersMetric() {
  await new Promise((r) => setTimeout(r, 1200));
  return <div className="p-4 border rounded-xl"><p className="text-2xl font-bold">1,234</p><p className="text-sm text-gray-500">Active users</p></div>;
}
async function RecentOrders() {
  await new Promise((r) => setTimeout(r, 2000));
  return <div className="p-4 border rounded-xl"><p className="font-semibold mb-2">Recent Orders</p><p className="text-sm text-gray-500">10 orders loaded</p></div>;
}
async function RevenueChart() {
  await new Promise((r) => setTimeout(r, 1500));
  return <div className="p-4 border rounded-xl h-[300px] flex items-center justify-center text-gray-400">Chart loaded</div>;
}

export default function AnalyticsPage() {
  // Each Suspense boundary resolves independently
  // The page is interactive immediately — slow sections stream in
  return (
    <div className="p-6 space-y-6">
      <h1 className="text-2xl font-bold">Analytics</h1>

      <div className="grid grid-cols-2 gap-4">
        <Suspense fallback={<CardSkeleton />}><RevenueMetric /></Suspense>
        <Suspense fallback={<CardSkeleton />}><UsersMetric /></Suspense>
      </div>

      <Suspense fallback={<ChartSkeleton />}><RevenueChart /></Suspense>

      <Suspense fallback={<TableSkeleton rows={10} />}><RecentOrders /></Suspense>
    </div>
  );
}
```

### 5. Lazy modal — only load when opened
```tsx
// src/components/LazyModal.tsx
"use client";

import dynamic from "next/dynamic";
import { useState } from "react";

// HeavyModal is only downloaded when isOpen becomes true
const HeavyModal = dynamic(() => import("./HeavyModal"), {
  ssr: false,
  loading: () => (
    <div className="fixed inset-0 bg-black/50 flex items-center justify-center">
      <div className="bg-white rounded-xl p-8 animate-pulse w-80 h-48" />
    </div>
  ),
});

export function LazyModal() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <button onClick={() => setIsOpen(true)}
        className="px-4 py-2 bg-blue-600 text-white rounded">
        Open Modal
      </button>
      {isOpen && <HeavyModal onClose={() => setIsOpen(false)} />}
    </>
  );
}
```
