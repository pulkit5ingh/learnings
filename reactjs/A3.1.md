## Goal
Set up client-side routing in a React app using React Router v6, including nested routes and active link highlighting.

## Background

### Client-Side Routing
In a single-page application (SPA), the browser loads one HTML page and JavaScript handles all navigation. React Router intercepts URL changes and renders different components without making a network request to the server. The URL still changes (so the browser back button works and links are shareable), but there's no full page reload.

### React Router v6 Mental Model
Everything is a `<Route>`. Routes render their `element` when the URL matches their `path`. Nested routes let a parent route render a layout (header, nav, sidebar) while the child routes render page content into an `<Outlet>` placeholder. `<Link>` renders an anchor that changes the URL without reloading. `<NavLink>` is a `<Link>` that knows whether it's currently active, letting you style it differently.

### v6 vs v5 Changes
In v6, you use `<Routes>` (plural) instead of `<Switch>`, route matching is exact by default (no `exact` prop needed), nested routes use `<Outlet>` instead of `{children}`, and the `component` prop is replaced by `element` (which takes JSX, not a component reference).

### Task 2 — Basic Routes

```tsx
// src/App.tsx
import { Routes, Route } from 'react-router-dom'
import { RootLayout } from './layouts/RootLayout'
import { HomePage } from './pages/HomePage'
import { AboutPage } from './pages/AboutPage'
import { ContactPage } from './pages/ContactPage'
import { NotFoundPage } from './pages/NotFoundPage'

export default function App() {
  return (
    <Routes>
      {/* Parent route renders RootLayout with <Outlet> */}
      <Route path="/" element={<RootLayout />}>
        {/* index route renders at exactly "/" */}
        <Route index element={<HomePage />} />
        <Route path="about" element={<AboutPage />} />
        <Route path="contact" element={<ContactPage />} />
        {/* Catch-all for unmatched URLs */}
        <Route path="*" element={<NotFoundPage />} />
      </Route>
    </Routes>
  )
}
```

### Task 3 — Layout with Outlet and NavLink

```tsx
// src/layouts/RootLayout.tsx
import { Outlet, NavLink, Link } from 'react-router-dom'

const navLinks = [
  { to: '/', label: 'Home', end: true },   // end=true: active only at exact /
  { to: '/about', label: 'About' },
  { to: '/contact', label: 'Contact' },
]

export function RootLayout() {
  return (
    <div style={{ minHeight: '100vh', display: 'flex', flexDirection: 'column' }}>
      <header style={{ background: '#1e293b', padding: '0 2rem' }}>
        <nav style={{ display: 'flex', alignItems: 'center', height: 60, gap: '0.5rem' }}>
          <Link to="/" style={{ color: 'white', fontWeight: 'bold', fontSize: 20, marginRight: 'auto', textDecoration: 'none' }}>
            My App
          </Link>

          {navLinks.map(link => (
            <NavLink
              key={link.to}
              to={link.to}
              end={link.end}
              style={({ isActive }) => ({
                color: isActive ? '#38bdf8' : '#94a3b8',
                textDecoration: 'none',
                padding: '6px 12px',
                borderRadius: 4,
                fontWeight: isActive ? 600 : 400,
                background: isActive ? 'rgba(56,189,248,0.1)' : 'transparent',
              })}
            >
              {link.label}
            </NavLink>
          ))}
        </nav>
      </header>

      <main style={{ flex: 1, padding: '2rem' }}>
        <Outlet />  {/* Child route renders here */}
      </main>

      <footer style={{ background: '#f1f5f9', borderTop: '1px solid #e2e8f0', padding: '1rem 2rem', textAlign: 'center', color: '#64748b', fontSize: 14 }}>
        © 2025 My App
      </footer>
    </div>
  )
}
```

// src/layouts/DashboardLayout.tsx
import { Outlet, NavLink } from 'react-router-dom'

const dashboardLinks = [
  { to: '/dashboard', label: '📊 Overview', end: true },
  { to: '/dashboard/users', label: '👥 Users' },

export function DashboardLayout() {
  return (
    <div style={{ display: 'flex', gap: '2rem', minHeight: 400 }}>

      <aside style={{ width: 200, flexShrink: 0 }}>
        <nav style={{ display: 'flex', flexDirection: 'column', gap: 4 }}>
          {dashboardLinks.map(link => (
            <NavLink
              key={link.to}
              to={link.to}
              end={link.end}
              style={({ isActive }) => ({
                display: 'block',
                padding: '8px 12px',
                borderRadius: 6,

                color: isActive ? '#2563eb' : '#374151',
                background: isActive ? '#eff6ff' : 'transparent',
                fontWeight: isActive ? 600 : 400,
              })}
            >
              {link.label}
            </NavLink>
          ))}
        </nav>
      </aside>

      {/* Nested routes render here */}
      <div style={{ flex: 1 }}>
        <Outlet />
      </div>
    </div>
  )
}
