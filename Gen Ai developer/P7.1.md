# P7.1 — AI Meeting Assistant

**Phase:** 7 — Multimodal AI  
**Topic:** Audio, Whisper, LLMs, PDF Generation  
**Type:** Project

---

## Goal
Build a complete meeting intelligence tool: record → transcribe → diarize → summarize → action items → formatted PDF report. This is a real enterprise product (Otter.ai, Fireflies.ai do exactly this).

---

## What to Build

### 1. Audio Input Support
Accept audio from multiple sources:
- Upload a file (MP3, MP4, WAV, M4A)
- Record directly in the browser (use Streamlit's audio_recorder component)
- Provide a YouTube/Zoom cloud recording URL (use `yt-dlp`)

### 2. Transcription Pipeline
- Transcribe with Whisper `medium` model for best accuracy
- Enable word-level timestamps
- Handle long recordings: split into 30-minute chunks, transcribe in parallel, stitch back
- Language detection: auto-detect, report detected language

### 3. Speaker Diarization
- Use `pyannote/speaker-diarization-3.1`
- Align diarization with Whisper timestamps
- Allow user to rename speakers: "Speaker 1 → Pulkit", "Speaker 2 → Rahul"
- Show speaker talk-time breakdown (who spoke the most)

### 4. LLM Intelligence Layer
Extract from the transcript:

```python
sections = {
    "executive_summary": "3-sentence overview of the meeting",
    "key_decisions": ["Decision 1: ...", "Decision 2: ..."],
    "action_items": [
        {"task": "...", "owner": "Speaker Name", "deadline": "...", "priority": "high|medium|low"}
    ],
    "key_discussions": [
        {"topic": "...", "summary": "...", "timestamp": "00:05:30"}
    ],
    "open_questions": ["Question that was raised but not resolved"],
    "next_meeting_agenda": ["Suggested agenda items for follow-up"]
}
```

### 5. PDF Report Generation
Use `reportlab` or `weasyprint` to generate a professional PDF:
```
[Company Logo / Meeting Title]
Meeting: Product Sync — June 4, 2025
Duration: 45 min | Participants: Pulkit, Rahul, Priya

EXECUTIVE SUMMARY
...

ACTION ITEMS
┌──────────────────────────┬──────────┬───────────┬──────────┐
│ Task                     │ Owner    │ Deadline  │ Priority │
├──────────────────────────┼──────────┼───────────┼──────────┤
│ Write design doc for API │ Pulkit   │ June 10   │ High     │
└──────────────────────────┴──────────┴───────────┴──────────┘

KEY DISCUSSIONS
...

FULL TRANSCRIPT
[00:00:00] Pulkit: Good morning everyone...
```

### 6. Streamlit Web App
- Upload audio or record in-browser
- Show real-time progress: "Transcribing... Diarizing... Analyzing..."
- Display results in tabs: Summary | Action Items | Transcript | Download PDF
- Speaker renaming UI
- Email integration: send PDF report to all participants via SendGrid API

### 7. Calendar Integration (Optional Stretch)
- Add detected action items directly to Google Calendar as tasks
- Use Google Calendar API with OAuth2
