# P7.3 — AI Product Photo Generator

**Phase:** 7 — Multimodal AI  
**Topic:** Image Generation, Stable Diffusion, ControlNet  
**Type:** Project

---

## Goal
Build a tool that takes a plain product photo and generates multiple professional lifestyle/context images using Stable Diffusion. Used in real e-commerce (Shopify, Amazon sellers) to create product listings without expensive photo shoots.

---

## What to Build

### 1. Input Processing
Accept a product photo (PNG/JPG):
- Remove background using `rembg`:
  ```python
  from rembg import remove
  from PIL import Image
  input_img = Image.open("product.jpg")
  clean_product = remove(input_img)  # transparent background PNG
  ```
- Enhance the product image: brightness, contrast, sharpen

### 2. Context/Scene Generation
For the background-removed product, generate lifestyle scenes:

**Method A — Inpainting**:
- Place the product on a simple background
- Use inpainting to generate a contextually appropriate scene around it

**Method B — ControlNet + img2img**:
- Use the product as a reference with ControlNet
- Generate new images where the product is in different environments

### 3. Scene Templates
Build a system of scene prompts:
```python
SCENE_TEMPLATES = {
    "modern_kitchen": "modern minimal kitchen counter, natural light, white marble, bokeh background",
    "outdoor_nature": "outdoor nature setting, green plants, sunlight filtering through leaves",
    "luxury_lifestyle": "luxury apartment, golden hour light, artistic composition",
    "studio_white": "professional studio, pure white background, soft box lighting, product photography",
    "urban_street": "urban street setting, brick wall background, lifestyle photography",
}
```

Generate 5 variations for each template using different seeds.

### 4. Streamlit App
```
[Upload Product Photo] → [Remove Background Preview]

Scene Selection:
○ Modern Kitchen  ○ Outdoor  ○ Luxury  ○ Studio  ○ Urban

Number of variations: [3] [5] [8]
Style: [Photorealistic] [Artistic] [Minimalist]

[Generate] → Progress bar

Output: 4x3 grid of generated images
[Download All as ZIP] [Download Individual]
```

### 5. Quality Control
After generation, automatically:
- Check image sharpness (reject blurry outputs)
- Check if product is clearly visible (use CLIP similarity between product and output)
- Filter out generations where the product looks distorted

### 6. Batch Processing
```bash
python product_generator.py \
  --input ./products/ \      # folder of product images
  --scenes kitchen,studio \  # which scenes to generate
  --variations 5 \
  --output ./generated/
```

Process 10 products, generate 5 scene variations each = 50 output images.

### 7. Before/After Showcase
Create a showcase HTML page (auto-generated) that shows:
- Original product photo
- All generated lifestyle images in a carousel
- Download button per image
