## Goal
Build a robust PII (Personally Identifiable Information) detection and redaction pipeline. Critical for any LLM app that processes user-submitted text — required for GDPR, HIPAA compliance.

---

## PII Categories to Detect
- **Names**: person names
- **Emails**: email addresses
- **Phone numbers**: Indian (+91) and international formats
- **Addresses**: street addresses, cities, PIN codes
- **IDs**: Aadhaar number, PAN card, passport number, credit card numbers
- **Dates of birth**: various formats
- **IP addresses**: IPv4, IPv6
- **URLs**: personal website or profile links

### Part 2 — NER-Based Detection
Use a pretrained NER model for names and addresses (harder to catch with regex):
```python
from transformers import pipeline

ner = pipeline("ner", model="dslim/bert-base-NER", aggregation_strategy="simple")

def detect_names_and_locations(text: str):
    entities = ner(text)
    return [e for e in entities if e["entity_group"] in ["PER", "LOC"]]
```
