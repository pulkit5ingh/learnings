# P6.1 — Autonomous Research Agent

**Phase:** 6 — AI Agents  
**Topic:** Agents, Web Research, LangGraph  
**Type:** Project

---

## Goal
Build an agent that takes a research topic, autonomously searches the web, reads multiple sources, synthesizes findings, and outputs a structured research report with citations. Used in real consulting, investment research, and academic workflows.

---

## What to Build

### 1. Research Pipeline
```
User provides: topic + research depth (quick/standard/deep)

Agent:
1. Generate 5-10 search queries from the topic
2. For each query: search web → get URLs
3. Visit each URL → extract clean text (remove ads, nav, footers)
4. Rank sources by relevance
5. Synthesize findings → write report
6. Format with citations
```

### 2. Tool Suite
```python
tools = [
    web_search(query: str) -> List[{"url", "title", "snippet"}],
    fetch_page(url: str) -> str,  # returns clean text from URL
    extract_key_facts(text: str, topic: str) -> List[str],  # LLM-powered
    check_contradiction(fact1: str, fact2: str) -> bool,  # detect conflicting info
]
```

### 3. LangGraph Workflow
```
[START]
→ Query Generator Node: generate 5-8 search queries
→ Search Node: execute all queries in parallel
→ Filter Node: deduplicate and rank URLs by relevance
→ Scrape Node: fetch top-10 pages
→ Extract Node: extract key facts per page
→ Contradiction Check: flag conflicting claims
→ Synthesize Node: write the report
→ [END]
```

### 4. Report Format
```markdown
# Research Report: {topic}
**Date**: {date}  **Sources**: {n}

## Executive Summary
[3-5 sentence overview]

## Key Findings
### Finding 1: [title]
[paragraph] [Smith, 2024] [Jones et al., 2023]

### Finding 2: [title]
...

## Conflicting Views
[if any sources contradict each other, present both sides]

## Limitations & Gaps
[what this research couldn't find]

## Sources
1. [Title](URL) — Author, Date, Relevance score
2. ...
```

### 5. Depth Modes
- **Quick** (2 min): 5 searches, top-5 pages, 500-word report
- **Standard** (5 min): 8 searches, top-10 pages, 1000-word report
- **Deep** (15 min): 15 searches, top-20 pages, 2000-word report with subsections

### 6. Quality Features
- **Source credibility scoring**: prefer .edu, .gov, established publications over random blogs
- **Recency filter**: prefer sources from last 2 years (configurable)
- **Contradiction detection**: if two sources disagree on a fact, flag it in the report
- **Confidence indicators**: mark facts as "widely reported", "single source", "conflicting"

### 7. Output Formats
```bash
python research_agent.py --topic "State of RAG in 2025" --depth standard --format markdown
python research_agent.py --topic "..." --format pdf
python research_agent.py --topic "..." --format json  # structured data
```
