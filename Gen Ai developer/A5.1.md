# A5.1 — RAG Pipeline From Scratch (No Frameworks)

**Phase:** 5 — Retrieval-Augmented Generation  
**Topic:** RAG Fundamentals  
**Type:** Assignment

---

## Goal
Build a complete RAG pipeline without LangChain or LlamaIndex. Implementing it from scratch forces you to understand every component. This is essential before using frameworks.

---

## Task

### Part 1 — Document Loading & Chunking
- Load a 20+ page PDF (use any technical paper, textbook chapter, or long article)
- Use `PyPDF2` or `pdfplumber` to extract raw text
- Implement fixed-size chunking:
  - `chunk_text(text, chunk_size=500, overlap=50)` — split into chunks of 500 characters with 50-character overlap
- Print: total chunks created, average chunk size, shortest and longest chunk

### Part 2 — Embedding & Storage
- Use `sentence-transformers` to embed all chunks:
  - Model: `sentence-transformers/all-MiniLM-L6-v2`
  - Embed in batches of 32
- Store in ChromaDB:
  ```python
  import chromadb
  client = chromadb.Client()
  collection = client.create_collection("my_rag")
  collection.add(documents=chunks, embeddings=embeddings, ids=[...])
  ```
- Print: total documents in collection, time to embed all chunks

### Part 3 — Retrieval
Implement `retrieve(query, top_k=5)`:
- Embed the query
- Query ChromaDB for top-k most similar chunks
- Return: list of `{"chunk": "...", "similarity": 0.87}`
- Test with 10 questions from the document — manually check if retrieved chunks are relevant

### Part 4 — Generation
Implement `answer(query)`:
- Retrieve top-5 chunks
- Build the prompt:
  ```
  Context:
  [chunk 1]
  [chunk 2]
  ...

  Question: {query}
  Answer based only on the context above. If the answer is not in the context, say "I don't know."
  ```
- Call Claude or GPT-4 with this prompt
- Return the answer

### Part 5 — Test & Analyze
- Create 10 test Q&A pairs from the document (you write the questions, you know the correct answers)
- Run your RAG on all 10
- For each: was the correct chunk retrieved? Was the answer correct?
- Document 2 failure cases: why did it fail?
