# A6.4 — Human-in-the-Loop Agent

**Phase:** 6 — AI Agents  
**Topic:** Human-in-the-Loop, Agent Safety  
**Type:** Assignment

---

## Goal
Build an agent that pauses before taking consequential actions and waits for human approval. This pattern is critical for any production agent that touches real systems.

---

## Scenario: File System & Email Agent
The agent can:
- Read files (safe, auto-approved)
- Search the web (safe, auto-approved)
- Write/delete files (requires human approval)
- Send emails (requires human approval)

---

## Task

### Part 1 — Define the Approval Framework
```python
from enum import Enum

class ActionRisk(Enum):
    LOW = "auto_approve"      # read, search
    MEDIUM = "request_approval"  # write, create
    HIGH = "require_approval"    # delete, send, deploy

TOOL_RISK = {
    "read_file": ActionRisk.LOW,
    "web_search": ActionRisk.LOW,
    "write_file": ActionRisk.MEDIUM,
    "delete_file": ActionRisk.HIGH,
    "send_email": ActionRisk.HIGH,
}
```

### Part 2 — Implement the Approval Flow
When the agent wants to take a MEDIUM or HIGH risk action:
1. **Pause** — don't execute yet
2. **Display** to user: "Agent wants to: DELETE file 'old_reports.csv'. Approve? [y/n/modify]"
3. Three responses:
   - `y` → execute the action
   - `n` → tell the agent the action was denied, let it continue with a different approach
   - `modify: [new instruction]` → user provides a modified version of the action

### Part 3 — Build the CLI Interface
```bash
$ python agent.py
Task: Organize my Downloads folder. Delete files older than 30 days, keep the rest.

Agent: I'll start by listing all files in Downloads...
[AUTO] read_file("/Users/pulkit/Downloads") → [file list]

Agent: I found 15 files older than 30 days. I'll delete them.

⚠️  APPROVAL REQUIRED
Action: delete_file
Files: report_2024_jan.pdf, old_invoice.pdf, ... (13 more)
Risk: HIGH — This cannot be undone

[y] Approve  [n] Deny  [m] Modify  [d] Show details
> d

Files to delete:
  - report_2024_jan.pdf (Jan 5, 2024, 2.3MB)
  - old_invoice.pdf (Dec 12, 2023, 0.5MB)
  ...

[y] Approve  [n] Deny  [m] Modify
> m
Modify: Only delete files smaller than 1MB

Agent: Understood. I'll only delete files smaller than 1MB...
```

### Part 4 — Approval History Log
- Log every approval request to a JSON file: timestamp, action, risk level, user decision, outcome
- After the session, print a summary: "5 actions auto-approved, 3 requested approval (2 approved, 1 denied), 0 errors"

### Part 5 — Test Scenarios
Run the agent on 3 test tasks:
1. "Read all .txt files in a folder and summarize them" — should be fully auto-approved
2. "Write a summary report to a new file" — should pause once for approval
3. "Delete all temp files and email me the report" — should pause twice (delete + email)

Document the full conversation trace for each.
