## What to Build

### 1. Documentation Scraper
- Scrape the entire docs site (respect robots.txt)
- Extract clean text: remove nav, footer, code examples (or keep code separately)
- Save each page as: `{"url": "...", "title": "...", "content": "...", "last_updated": "..."}`
- Handle pagination, dynamic JS-rendered content (use Playwright if needed)

### 2. Ingestion Pipeline
- Semantic chunking with metadata preservation:
  - Each chunk stores: `source_url`, `page_title`, `section_heading`, `chunk_index`
- Embedding: `text-embedding-3-large` (OpenAI) or `BAAI/bge-large-en-v1.5` (open source)
- Store in Qdrant (production-ready vector DB):
  ```python
  # Run Qdrant locally with Docker
  docker run -p 6333:6333 qdrant/qdrant
  ```

### 3. Hybrid Search
Implement two retrievers and merge results:
- **Dense retrieval**: Qdrant vector search (semantic)
- **Sparse retrieval**: BM25 via `rank_bm25` library (keyword)
- **Reciprocal Rank Fusion (RRF)**: merge the two ranked lists
  ```python
  def rrf(dense_results, sparse_results, k=60):
      scores = {}
      for rank, doc in enumerate(dense_results):
          scores[doc.id] = scores.get(doc.id, 0) + 1 / (k + rank + 1)
      for rank, doc in enumerate(sparse_results):
          scores[doc.id] = scores.get(doc.id, 0) + 1 / (k + rank + 1)
      return sorted(scores.items(), key=lambda x: x[1], reverse=True)
  ```

### 4. Reranking
- Use Cohere Rerank API or cross-encoder to rerank top-20 → top-5

### 5. Generation with Streaming
- FastAPI endpoint with Server-Sent Events (SSE) for streaming:
  ```
  POST /ask
  Body: { "question": "...", "session_id": "..." }
  Response: SSE stream of tokens
  ```
- System prompt: answer using docs, cite source URLs
- Multi-turn: store last 5 turns in Redis by session_id

### 6. RAGAS Evaluation Suite
- Build 100-question benchmark against the documentation
- Run RAGAS: faithfulness, context precision, context recall
- Must hit: faithfulness >0.85, context precision >0.75

### 7. Observability
- Log every query to a PostgreSQL database: question, retrieved chunks, answer, latency, token count
- LangSmith integration for trace-level debugging
