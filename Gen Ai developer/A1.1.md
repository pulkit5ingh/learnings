# A1.1 — Gradient Descent From Scratch

**Phase:** 1 — Foundations  
**Topic:** Mathematics & Optimization  
**Type:** Assignment

---

## Goal
Implement gradient descent manually to minimize a quadratic function. Understand how learning rate affects convergence.

---

## Task

### Part 1 — Minimize a simple function
- Define `f(x) = (x - 3)^2 + 5`
- Compute the gradient analytically: `f'(x) = 2(x - 3)`
- Implement gradient descent loop:
  - Start at `x = 10`
  - Learning rates to test: `0.001`, `0.01`, `0.1`, `0.9`
  - Run for 100 iterations each

### Part 2 — Extend to 2D
- Define `f(x, y) = (x - 2)^2 + (y + 1)^2`
- Implement gradient descent in 2D
- Start at `(x=8, y=8)`, learning rate `0.1`, 200 iterations

### Part 3 — Visualize
- Plot the loss curve (f(x) vs iteration) for each learning rate
- Plot the 2D path of `(x, y)` as it converges (scatter plot with arrows)

---

## Requirements
- Use only `numpy` and `matplotlib`. No PyTorch or sklearn.
- Write as a single Python script or Jupyter notebook.

---

## Deliverables
- [ ] Working gradient descent for 1D and 2D functions
- [ ] Loss curve plots for all 4 learning rates
- [ ] 2D convergence path plot
- [ ] Short written observation: what happens when learning rate is too high (0.9)?
