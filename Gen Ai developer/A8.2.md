## Task

### Part 1 — FastAPI Server with Streaming
```python
from fastapi import FastAPI
from fastapi.responses import StreamingResponse

app = FastAPI()

@app.post("/v1/chat/completions")
async def chat(request: ChatRequest):
    async def generate():
        async for chunk in stream_from_llm(request.messages):
            yield f"data: {chunk.model_dump_json()}\n\n"
        yield "data: [DONE]\n\n"
    return StreamingResponse(generate(), media_type="text/event-stream")
```

Make it compatible with OpenAI's API format (same request/response schema):
- Accept `model`, `messages`, `temperature`, `max_tokens`, `stream`
- Return `ChatCompletionChunk` objects on stream

### Part 2 — API Key Authentication
- Store API keys in SQLite: `api_keys(key_hash, user_id, created_at, is_active)`
- Middleware that checks `Authorization: Bearer sk-...` header on every request
- Hash the key with SHA-256 before storing — never store raw keys
- Return 401 if key is missing, 403 if key is inactive

### Part 3 — Rate Limiting with Redis
```python
import redis
r = redis.Redis()

def check_rate_limit(user_id: str, limit: int = 100, window: int = 3600):
    key = f"ratelimit:{user_id}:{int(time.time() // window)}"
    current = r.incr(key)
    if current == 1:
        r.expire(key, window)
    if current > limit:
        raise HTTPException(429, "Rate limit exceeded")
    return {"remaining": limit - current}
```

Implement:
- Per-user rate limit: 100 requests/hour
- Per-IP rate limit: 20 requests/minute (for unauthenticated)
- Return `X-RateLimit-Remaining` and `X-RateLimit-Reset` headers

### Part 4 — Request Logging
Log every request to PostgreSQL:
```sql
CREATE TABLE request_logs (
    id SERIAL PRIMARY KEY,
    user_id TEXT,
    endpoint TEXT,
    model TEXT,
    prompt_tokens INT,
    completion_tokens INT,
    latency_ms INT,
    status_code INT,
    created_at TIMESTAMP DEFAULT NOW()
);
```

### Part 5 — Admin Endpoints
```
GET /admin/usage?user_id=xxx&start=2025-01-01&end=2025-06-01

```

Protect admin endpoints with a separate admin API key.
