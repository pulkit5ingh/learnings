# A1.3 — Cosine Similarity & Word Embeddings Visualization

**Phase:** 1 — Foundations  
**Topic:** Linear Algebra  
**Type:** Assignment

---

## Goal
Understand how vector similarity works by simulating word embeddings and visualizing their relationships using cosine similarity and t-SNE.

---

## Task

### Part 1 — Cosine Similarity From Scratch
- Implement cosine similarity: `cos(A, B) = (A · B) / (||A|| * ||B||)`
- Assign 10 words to 3D random vectors (use `np.random.seed(42)` for reproducibility)
- Compute the full 10x10 cosine similarity matrix
- Display it as a heatmap using seaborn

### Part 2 — Nearest Neighbors
- For each word, find its 3 most similar words based on cosine similarity
- Print a table: `word → [nearest1, nearest2, nearest3]`

### Part 3 — Visualize with t-SNE
- Use 50-dimensional random vectors for 20 words (grouped into 4 semantic groups of 5 words each — e.g., animals, fruits, vehicles, sports)
- Apply t-SNE to reduce to 2D
- Plot a scatter chart with word labels. Words in the same group should cluster together if t-SNE works correctly.

---

## Requirements
- Use `numpy`, `matplotlib`, `seaborn`, `sklearn.manifold.TSNE`
- Words must be manually assigned to groups (don't use real pre-trained embeddings)

---

## Deliverables
- [ ] Cosine similarity implemented from scratch (verified against `np.dot` baseline)
- [ ] Heatmap of 10x10 similarity matrix
- [ ] Nearest neighbors table
- [ ] t-SNE scatter plot with grouped word labels
- [ ] Written observation: do the 4 groups cluster visually?
