**Phase:** 3 — Natural Language Processing  
**Topic:** Transformers Architecture

## Goal
Implement the attention mechanism from scratch in PyTorch — the core building block of every modern LLM. Understand what Q, K, V actually mean and how masking works.

---

## Task

### Part 1 — Scaled Dot-Product Attention
Implement the function:
```python
def scaled_dot_product_attention(Q, K, V, mask=None):
    # Q: (batch, heads, seq_len, d_k)
    # K: (batch, heads, seq_len, d_k)
    # V: (batch, heads, seq_len, d_v)
    # Returns: (batch, heads, seq_len, d_v), attention_weights
    ...
```
- Compute `scores = QK^T / sqrt(d_k)`
- If mask provided, set masked positions to `-inf` before softmax
- Apply softmax to get attention weights
- Compute output: `attention_weights @ V`

### Part 2 — Multi-Head Attention
Implement `class MultiHeadAttention(nn.Module)`:
- Parameters: `d_model`, `num_heads`
- Project Q, K, V with separate linear layers
- Split into heads, apply attention, concatenate, project output
- Must match `nn.MultiheadAttention` output shape

### Part 3 — Verify Your Implementation
- Create a toy sequence: batch=2, seq_len=10, d_model=64, num_heads=4
- Run your implementation
- Run `nn.MultiheadAttention` on the same input (with same weights)
- Assert outputs are numerically close

### Part 4 — Causal (Decoder) Mask
- Create a causal mask: lower triangular matrix (each token can only attend to past tokens)
- Apply it to your attention function
- Visualize the attention weights as a heatmap for a 10-token sequence
- Verify that token 3 has zero attention weight on tokens 4-10

### Part 5 — Visualize Attention
- Use a pretrained `bert-base-uncased` and the `bertviz` library
- Visualize attention patterns for the sentence: "The cat sat on the mat because it was tired."
- Which head captures subject-verb relationships?
