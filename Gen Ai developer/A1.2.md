# A1.2 — Logistic Regression From Scratch

**Phase:** 1 — Foundations  
**Topic:** Mathematics & Statistics  
**Type:** Assignment

---

## Goal
Build logistic regression using only NumPy. No sklearn for training. Understand sigmoid, cross-entropy loss, and weight updates manually.

---

## Task

### Part 1 — Implement Logistic Regression
- Implement the sigmoid function: `σ(z) = 1 / (1 + e^-z)`
- Implement binary cross-entropy loss
- Implement gradient computation for weights `W` and bias `b`
- Implement the training loop (gradient descent)

### Part 2 — Train on Iris Dataset
- Load the Iris dataset from `sklearn.datasets` (only for loading, not training)
- Use only 2 classes (binary classification: class 0 vs class 1)
- Split: 80% train, 20% test (implement the split manually using numpy)
- Train for 1000 iterations, learning rate = 0.1

### Part 3 — Evaluate
- Compute accuracy, precision, recall, and F1 score manually (no sklearn metrics)
- Print a confusion matrix

### Part 4 — Compare
- Now train sklearn's `LogisticRegression` on the same data
- Compare your accuracy vs sklearn's accuracy

---

## Requirements
- No sklearn for training or evaluation — only for loading the dataset
- All math implemented in pure NumPy

- [ ] Loss curve plot (loss vs epoch)
