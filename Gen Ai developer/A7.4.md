## Task

### Part 1 — Basic Stable Diffusion
Load `stable-diffusion-v1-5` or `stabilityai/stable-diffusion-xl-base-1.0`:
```python
from diffusers import StableDiffusionPipeline
pipe = StableDiffusionPipeline.from_pretrained("runwayml/stable-diffusion-v1-5")
```

Generate the same scene with 5 different prompt styles:
1. Minimal: `"a cat"`
2. Descriptive: `"a fluffy orange tabby cat sitting on a windowsill"`
3. Style-specified: `"a fluffy orange tabby cat, oil painting, impressionist, warm colors"`
4. Negative prompt: same as 3, but add negative_prompt: `"blurry, low quality, distorted"`
5. Reference artist: `"a fluffy orange tabby cat, in the style of Studio Ghibli, soft light"`

For each: show the generated image + the exact prompt used.

### Part 2 — Sampling Parameters Experiment
Using the same prompt, vary:
- **Guidance scale**: 3, 7, 12, 20 (how strongly to follow the prompt)
- **Steps**: 10, 20, 30, 50 (quality vs speed)
- **Seed**: 42, 123, 999 (how much results vary)

Create a grid image: rows = guidance scale, columns = steps. Write your observations.

### Part 3 — ControlNet
Install `controlnet_aux` and use ControlNet to condition on a structure:

**Experiment 1 — Canny Edge Control**: 
- Take a real photo
- Extract Canny edges
- Use ControlNet-Canny to generate a new image that follows the same structure but with a different style
- Show: original → edges → generated

**Experiment 2 — Pose Control**:
- Take a person image
- Extract human pose skeleton
- Generate a new character in the same pose with a different appearance

**Experiment 3 — Depth Control**:
- Take any scene photo
- Extract depth map
- Generate a fantasy version of the scene preserving the spatial layout

### Part 4 — img2img
Use the img2img pipeline to:
- Take any photo
- Generate 5 variations with different strength values (0.3, 0.5, 0.7, 0.9)
- Show how strength controls how much the original image is preserved

### Part 5 — Prompt Engineering Guide
Based on everything you learned, write a structured guide:
- 5 elements of a good prompt (subject, style, lighting, camera, quality keywords)
- 10 prompt modifiers that consistently improve quality
- 5 negative prompt terms that always help
- ControlNet use cases: when to use canny vs pose vs depth
- Common failure modes and fixes
