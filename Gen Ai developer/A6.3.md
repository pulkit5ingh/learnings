# A6.3 — Multi-Step LangGraph Workflow

**Phase:** 6 — AI Agents  
**Topic:** LangGraph, Agent Orchestration  
**Type:** Assignment

---

## Goal
Build a LangGraph workflow for a multi-step content creation task. Learn how to design stateful graphs with conditional branching — the foundation of production agentic systems.

---

## Workflow: Research → Outline → Draft → Review → Publish

```
[START] 
  → Research Node (search web, find sources)
  → Outline Node (structure the content)
  → Draft Node (write the full article)
  → Review Node (check quality, decide: pass or revise)
       ↓ FAIL (quality too low)
  → Revise Node (address review feedback)
       ↓ back to Review
  → [END] (when review passes)
```

## Task

### Part 1 — Define State
```python
from typing import TypedDict, List

class ArticleState(TypedDict):
    topic: str
    sources: List[str]        # URLs + summaries from research
    outline: str              # structured outline
    draft: str                # full article text
    review_feedback: str      # reviewer's comments
    revision_count: int       # how many revisions happened
    quality_score: float      # 1-10 from reviewer
    final_article: str
```

def should_revise(state: ArticleState) -> str:
    # Conditional edge function
    if state["quality_score"] >= 7.5 or state["revision_count"] >= 3:

### Part 3 — Build the Graph
```python
from langgraph.graph import StateGraph

workflow = StateGraph(ArticleState)
workflow.add_node("research", research_node)
workflow.add_node("outline", outline_node)
# ... add all nodes

# Add edges
workflow.add_edge("research", "outline")
workflow.add_conditional_edges("review", should_revise, {"revise": "revise", "end": END})

app = workflow.compile()
```
