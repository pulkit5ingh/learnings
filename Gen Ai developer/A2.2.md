# A2.2 — CNN for CIFAR-10 Classification in PyTorch

**Phase:** 2 — Machine Learning Core  
**Topic:** Deep Learning, CNNs  
**Type:** Assignment

---

## Goal
Build a CNN in PyTorch from scratch. Learn how to structure a training loop, apply data augmentation, and diagnose overfitting.

---

## Task

### Part 1 — Build the CNN
Define a `CNN` class using `nn.Module` with this architecture:
```
Conv2d(3, 32, 3, padding=1) → BatchNorm → ReLU → MaxPool
Conv2d(32, 64, 3, padding=1) → BatchNorm → ReLU → MaxPool
Conv2d(64, 128, 3, padding=1) → BatchNorm → ReLU → MaxPool
Flatten
Linear(128*4*4, 256) → ReLU → Dropout(0.5)
Linear(256, 10)
```

### Part 2 — Data Pipeline
- Use `torchvision.datasets.CIFAR10`
- Training augmentations: `RandomHorizontalFlip`, `RandomCrop(32, padding=4)`, `ColorJitter`
- Validation: only `ToTensor` + `Normalize`

### Part 3 — Training Loop
- Optimizer: Adam, lr=0.001
- Loss: CrossEntropyLoss
- Train for 30 epochs
- Log training loss and validation accuracy per epoch
- Save the best model checkpoint based on validation accuracy

### Part 4 — Evaluate
- Load the best checkpoint, evaluate on the test set
- Plot the training loss curve and validation accuracy curve
- Show a confusion matrix for all 10 CIFAR-10 classes
- Visualize 5 correctly classified and 5 misclassified examples

### Part 5 — Experiment
- Remove BatchNorm and retrain. Compare accuracy and convergence speed.
- Write a 3-sentence observation on what BatchNorm does.
