# P1.2 — EDA Dashboard on Real Dataset

**Phase:** 1 — Foundations  
**Topic:** Python for AI, Data Analysis  
**Type:** Project

---

## Goal
Perform a complete exploratory data analysis (EDA) on a real-world dataset. Build a structured, reproducible analysis notebook that tells a story with data.

---

## Dataset Options (pick one)
- Kaggle House Prices (regression target)
- Titanic survival dataset (classification target)
- NYC Taxi Trip Duration
- World Happiness Report

---

## What to Build

### 1. Data Loading & Overview
- Load dataset with Pandas
- Print shape, dtypes, first 5 rows
- Count missing values per column
- Count duplicate rows

### 2. Missing Value Analysis
- Visualize missing values as a heatmap (seaborn)
- Decide and implement: drop, fill with mean/median/mode, or flag as a new category
- Document every decision and why

### 3. Univariate Analysis
- For each numerical column: histogram + KDE plot
- For each categorical column: bar chart of value counts
- Identify skewed distributions

### 4. Bivariate Analysis
- Correlation heatmap of all numerical features
- Scatter plots for top 5 features most correlated with the target
- Box plots for categorical features vs the target

### 5. Feature Insights
- Identify top 3 features most predictive of the target (based on correlation or visual analysis)
- Identify any data quality issues found (outliers, weird values, suspicious distributions)
- Write a 1-paragraph summary of your key findings

### 6. Outlier Detection
- Use IQR method to identify outliers in the top features
- Visualize with box plots
- Document how you handle them

- [ ] All plots properly labeled (titles, axes, legends)

- [ ] Written summary of 3 key insights from the data
