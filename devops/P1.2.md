# P1.2 — Automated Server Provisioning Script

**Phase:** 1 — Linux & Networking Fundamentals  
**Topic:** Bash automation, server setup  
**Type:** Project

---

## Goal
Build a production-grade server provisioning script that sets up a fresh Ubuntu server from scratch — the kind of script you'd use to bootstrap 100 identical servers or recover from a disaster.

---

## What to Build
A script that takes a fresh Ubuntu 22.04 server and configures it as a secure, monitored web server with:
- Hardened SSH
- UFW firewall
- Nginx + HTTPS with self-signed cert
- Fail2ban
- Automatic security updates
- System monitoring (netdata or custom scripts)
- Swap file configuration
- NTP time sync
- User accounts for team members

## Architecture
```
provision.sh         — main orchestrator
config/
  users.conf         — list of users to create with their SSH keys
  packages.list      — packages to install
  nginx-template.conf — nginx config template
modules/
  00-update.sh       — system update
  01-users.sh        — create users, configure SSH keys
  02-ssh-hardening.sh — harden SSH config
  03-firewall.sh     — UFW rules
  04-nginx.sh        — install and configure nginx
  05-fail2ban.sh     — install and configure fail2ban
  06-security.sh     — automatic updates, auditing
  07-monitoring.sh   — install monitoring
  08-verify.sh       — run verification checks
```

## Key Features

### Idempotent Design
Running the script twice should not break anything:
```bash
# Instead of: apt-get install nginx
# Use: apt-get install -y nginx  (already installed = no-op)

# Instead of: useradd alice
# Use:
if ! id alice &>/dev/null; then
    useradd -m -s /bin/bash alice
fi

# Instead of: ufw allow 80
# Check first:
ufw status | grep -q "80/tcp" || ufw allow 80/tcp
```

### Verification Module
```bash
# 08-verify.sh
verify() {
    local test_name=$1
    local command=$2
    
    if eval "$command" &>/dev/null; then
        echo "✓ $test_name"
        return 0
    else
        echo "✗ FAILED: $test_name"
        FAILURES=$((FAILURES + 1))
        return 1
    fi
}

verify "nginx is running"     "systemctl is-active nginx"
verify "SSH port 22 is open"  "ss -tlnp | grep ':22 '"
verify "UFW is enabled"       "ufw status | grep -q 'Status: active'"
verify "fail2ban running"     "systemctl is-active fail2ban"
verify "HTTP responds"        "curl -sf http://localhost"
verify "Swap is configured"   "swapon --show | grep -q '/swapfile'"
```

### User Management from Config
```bash
# config/users.conf
# format: username:ssh_public_key_file
alice:keys/alice.pub
bob:keys/bob.pub

# modules/01-users.sh
while IFS=: read -r username keyfile; do
    [[ "$username" =~ ^#|^$ ]] && continue
    
    if ! id "$username" &>/dev/null; then
        useradd -m -s /bin/bash -G sudo "$username"
        log "Created user: $username"
    fi
    
    mkdir -p "/home/$username/.ssh"
    cat "$keyfile" >> "/home/$username/.ssh/authorized_keys"
    chmod 700 "/home/$username/.ssh"
    chmod 600 "/home/$username/.ssh/authorized_keys"
    chown -R "$username:$username" "/home/$username/.ssh"
done < config/users.conf
```
