# P3.1 — Containerize a Full-Stack Web Application

**Phase:** 3 — Containers (Docker)  
**Topic:** Docker, Docker Compose, production containerization  
**Type:** Project

---

## Goal
Take a real full-stack application (frontend + backend + database + cache) and containerize it for both local development and production deployment.

---

## The Application
Build a simple Blog platform:
- **Frontend**: React or Next.js (served by nginx in production)
- **Backend**: FastAPI or Node.js (REST API)
- **Database**: PostgreSQL
- **Cache**: Redis (for session storage and caching)
- **Reverse Proxy**: nginx (routes /api/* to backend, /* to frontend)

## Architecture
```
Browser → nginx:80
              ├── /         → frontend (React build, served by nginx)
              ├── /api/*    → api:8080 (FastAPI)
              └── /static/* → nginx (static files directly)

api → PostgreSQL (postgres:5432)
api → Redis (redis:6379)
```

## Dockerfile Strategy

### Frontend (multi-stage)
```dockerfile
# Build stage
FROM node:20-alpine AS builder
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

# Production stage
FROM nginx:alpine AS runner
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx/spa.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
```

### Backend (multi-stage, non-root)
```dockerfile
FROM python:3.11-slim AS builder
WORKDIR /build
COPY requirements.txt .
RUN pip install --user --no-cache-dir -r requirements.txt

FROM python:3.11-slim AS runtime
RUN useradd --uid 1001 --no-create-home --shell /sbin/nologin appuser
WORKDIR /app
COPY --from=builder /root/.local /home/appuser/.local
COPY --chown=appuser:appuser src/ ./src/
USER appuser
ENV PATH=/home/appuser/.local/bin:$PATH
HEALTHCHECK --interval=30s --timeout=3s \
    CMD python -c "import urllib.request; urllib.request.urlopen('http://localhost:8080/health')"
CMD ["uvicorn", "src.main:app", "--host", "0.0.0.0", "--port", "8080"]
```

## Docker Compose: Development vs Production

### Development compose
```yaml
# docker-compose.dev.yml
services:
  frontend:
    build:
      target: builder
    volumes:
      - ./frontend/src:/app/src  # hot reload
    command: npm run dev
    ports:
      - "3000:3000"

  api:
    build:
      target: builder
    volumes:
      - ./backend/src:/app/src  # hot reload
    command: uvicorn src.main:app --host 0.0.0.0 --port 8080 --reload
    environment:
      - DEBUG=true
      - DATABASE_URL=postgresql://user:pass@db:5432/blogdb
```

### Production compose
```yaml
# docker-compose.prod.yml
services:
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx/prod.conf:/etc/nginx/nginx.conf:ro
      - certbot_data:/etc/letsencrypt:ro
    depends_on:
      - frontend
      - api

  frontend:
    image: ghcr.io/myorg/blog-frontend:${TAG}
    deploy:
      replicas: 1

  api:
    image: ghcr.io/myorg/blog-api:${TAG}
    env_file: .env.production
    deploy:
      replicas: 2
      resources:
        limits:
          cpus: '0.5'
          memory: 512M

  db:
    image: postgres:16-alpine
    volumes:
      - postgres_data:/var/lib/postgresql/data
    env_file: .env.production
    deploy:
      resources:
        limits:
          memory: 1G
```

## nginx Configuration
```nginx
# nginx/prod.conf
upstream api {
    server api:8080;
}

server {
    listen 80;
    server_name myapp.com;

    # Serve React app for all routes (SPA routing)
    root /usr/share/nginx/html;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # Proxy API requests
    location /api/ {
        proxy_pass http://api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_read_timeout 30s;
    }

    # Security headers
    add_header X-Content-Type-Options nosniff;
    add_header X-Frame-Options DENY;
    add_header Strict-Transport-Security "max-age=31536000" always;

    # Gzip
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
```
