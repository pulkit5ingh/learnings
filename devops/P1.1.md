# P1.1 ‚Äî System Monitoring Dashboard (Bash)

**Phase:** 1 ‚Äî Linux & Networking Fundamentals  
**Topic:** Shell scripting, system monitoring  
**Type:** Project

---

## Goal
Build a terminal-based system monitoring dashboard in pure bash. No external tools ‚Äî just the Linux filesystem, `/proc`, and shell scripting.

---

## What to Build
A dashboard that refreshes every 5 seconds and shows:
- CPU usage (per-core bars and average)
- Memory usage (used/total with bar)
- Disk usage per mounted filesystem
- Top 5 processes by CPU
- Top 5 processes by memory
- Network I/O (bytes sent/received since last refresh)
- Last 5 system log entries

## Key Implementation Details

### CPU Usage from /proc
```bash
# /proc/stat format:
# cpu  user nice system idle iowait irq softirq steal guest
get_cpu_usage() {
    local cpu1 cpu2
    cpu1=($(grep '^cpu ' /proc/stat))
    sleep 0.5
    cpu2=($(grep '^cpu ' /proc/stat))
    
    local total1=$(( cpu1[1]+cpu1[2]+cpu1[3]+cpu1[4]+cpu1[5]+cpu1[6]+cpu1[7] ))
    local total2=$(( cpu2[1]+cpu2[2]+cpu2[3]+cpu2[4]+cpu2[5]+cpu2[6]+cpu2[7] ))
    local idle1=${cpu1[4]}
    local idle2=${cpu2[4]}
    
    local total_diff=$(( total2 - total1 ))
    local idle_diff=$(( idle2 - idle1 ))
    
    echo $(( 100 * (total_diff - idle_diff) / total_diff ))
}
```

### Progress Bar Function
```bash
draw_bar() {
    local value=$1   # 0-100
    local width=${2:-50}
    local fill=$(( value * width / 100 ))
    local empty=$(( width - fill ))
    
    printf '['
    printf '%0.s#' $(seq 1 $fill)
    printf '%0.s-' $(seq 1 $empty)
    printf '] %3d%%' $value
}
```

### Color Output
```bash
RED='\033[0;31m'
YELLOW='\033[1;33m'
GREEN='\033[0;32m'
NC='\033[0m'

colorize_usage() {
    local value=$1
    if   [ $value -ge 90 ]; then echo -e "${RED}${value}%${NC}"
    elif [ $value -ge 70 ]; then echo -e "${YELLOW}${value}%${NC}"
    else                         echo -e "${GREEN}${value}%${NC}"
    fi
}
```
