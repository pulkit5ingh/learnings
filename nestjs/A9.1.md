## Goal
Add real-time bidirectional communication to a NestJS app using WebSocket gateways, implement rooms, authenticate WebSocket connections using JWT, and understand multi-node scaling considerations.

## Background

NestJS WebSocket gateways wrap either `socket.io` (default) or `ws`. The `@WebSocketGateway()` decorator creates a WebSocket server alongside the HTTP server. `@SubscribeMessage('event')` handles incoming events from clients.

**Rooms** (socket.io concept) allow broadcasting to a subset of connected clients. A user joins a room, and any event emitted to that room goes to all clients in it.

**Authentication** over WebSocket is done during the connection handshake. The client sends a JWT in the `auth` object of the socket.io handshake. The gateway validates it using the same `JwtService` as your HTTP guards.

**Scaling** is the hard part: when running multiple server instances, each instance has its own in-memory socket.io store. A message emitted on instance A doesn't reach clients connected to instance B. Solution: the `socket.io-redis` (or `@socket.io/redis-adapter`) package syncs events across instances via Redis pub/sub.

### 2. Basic WebSocket gateway

**`src/chat/chat.gateway.ts`**
```typescript
import {
  ConnectedSocket,
  MessageBody,
  OnGatewayConnection,
  OnGatewayDisconnect,
  OnGatewayInit,
  SubscribeMessage,
  WebSocketGateway,
  WebSocketServer,
  WsException,
} from '@nestjs/websockets';
import { Logger, UseGuards } from '@nestjs/common';
import { Server, Socket } from 'socket.io';
import { WsJwtGuard } from './guards/ws-jwt.guard';

@WebSocketGateway({
  cors: {
    origin: ['http://localhost:3001'],
    credentials: true,
  },
  namespace: '/chat',   // optional: separate namespace
})
export class ChatGateway implements OnGatewayInit, OnGatewayConnection, OnGatewayDisconnect {
  @WebSocketServer()
  server: Server;

  private readonly logger = new Logger(ChatGateway.name);

  afterInit(server: Server) {
    this.logger.log('WebSocket Gateway initialized');
  }

  async handleConnection(client: Socket) {
    const user = client.handshake.auth.user;
    if (!user) {
      client.disconnect();
      return;
    }
    this.logger.log(`Client connected: ${client.id} (user: ${user.id})`);
    // Join a room for this user (for direct messages)
    await client.join(`user:${user.id}`);
  }

  handleDisconnect(client: Socket) {
    this.logger.log(`Client disconnected: ${client.id}`);
  }

  // Client sends: { room: 'general', message: 'Hello!' }
  @SubscribeMessage('join_room')
  async handleJoinRoom(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { room: string },
  ) {
    await client.join(data.room);
    this.logger.log(`Client ${client.id} joined room: ${data.room}`);
    // Notify the room that someone joined
    this.server.to(data.room).emit('user_joined', {
      room: data.room,
      userId: (client.handshake.auth.user as any)?.id,
    });
    return { event: 'joined', room: data.room };
  }

  @SubscribeMessage('leave_room')
  async handleLeaveRoom(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { room: string },
  ) {
    await client.leave(data.room);
    this.server.to(data.room).emit('user_left', {
      room: data.room,
      userId: (client.handshake.auth.user as any)?.id,
    });
  }

  @SubscribeMessage('send_message')
  @UseGuards(WsJwtGuard)
  async handleMessage(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { room: string; content: string },
  ) {
    const user = (client as any).user;

    const message = {
      id: Date.now().toString(),
      room: data.room,
      content: data.content,
      sender: { id: user.id, name: user.name },
      timestamp: new Date().toISOString(),
    };

    // Broadcast to all in the room (including sender)
    this.server.to(data.room).emit('new_message', message);

    return message;   // returned to the sender as acknowledgement
  }

  @SubscribeMessage('direct_message')
  @UseGuards(WsJwtGuard)
  async handleDirectMessage(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { recipientId: number; content: string },
  ) {
    const sender = (client as any).user;

    const message = {
      id: Date.now().toString(),
      content: data.content,
      sender: { id: sender.id, name: sender.name },
      timestamp: new Date().toISOString(),
    };

    // Send to recipient's room (they joined it on connect)
    this.server.to(`user:${data.recipientId}`).emit('direct_message', message);
    return message;
  }
}
```

### 3. WebSocket JWT guard

**`src/chat/guards/ws-jwt.guard.ts`**
```typescript
import { CanActivate, ExecutionContext, Injectable } from '@nestjs/common';
import { JwtService } from '@nestjs/jwt';
import { ConfigService } from '@nestjs/config';
import { WsException } from '@nestjs/websockets';
import { Socket } from 'socket.io';

@Injectable()
export class WsJwtGuard implements CanActivate {
  constructor(
    private readonly jwtService: JwtService,
    private readonly config: ConfigService,
  ) {}

  canActivate(context: ExecutionContext): boolean {
    const client: Socket = context.switchToWs().getClient();
    const token = client.handshake.auth.token as string;

    if (!token) throw new WsException('Missing authentication token');

    try {
      const payload = this.jwtService.verify(token, {
        secret: this.config.get<string>('JWT_SECRET'),
      });
      (client as any).user = payload;
      return true;
    } catch {
      throw new WsException('Invalid or expired token');
    }
  }
}
```

### 4. JWT verification during connection handshake

For gateway-level authentication (reject on connect, not per-event):

```typescript
import * as jwt from 'jsonwebtoken';

@WebSocketGateway()
export class ChatGateway implements OnGatewayConnection {
  constructor(
    private readonly jwtService: JwtService,
    private readonly usersService: UsersService,
    private readonly config: ConfigService,
  ) {}

  async handleConnection(client: Socket) {
    const token = client.handshake.auth.token as string;

    if (!token) {
      client.emit('error', { message: 'No token provided' });
      client.disconnect(true);
      return;
    }

    try {
      const payload = this.jwtService.verify<JwtPayload>(token, {
        secret: this.config.get('JWT_SECRET'),
      });
      const user = await this.usersService.findOne(payload.sub);
      (client as any).user = user;
      this.logger.log(`Authenticated: ${user.email}`);
    } catch (err) {
      client.emit('error', { message: 'Authentication failed' });
      client.disconnect(true);
    }
  }
}
```

### 5. Presence tracking

```typescript
@Injectable()
export class PresenceService {
  private online = new Map<number, string[]>();  // userId → socketIds

  userOnline(userId: number, socketId: string): void {
    const sockets = this.online.get(userId) ?? [];
    sockets.push(socketId);
    this.online.set(userId, sockets);
  }

  userOffline(userId: number, socketId: string): void {
    const sockets = (this.online.get(userId) ?? []).filter(id => id !== socketId);
    if (sockets.length === 0) this.online.delete(userId);
    else this.online.set(userId, sockets);
  }

  isOnline(userId: number): boolean {
    return this.online.has(userId);
  }

  getOnlineUsers(): number[] {
    return Array.from(this.online.keys());
  }
}
```

### 6. Redis adapter for multi-node scaling

**`src/main.ts`**
```typescript
import { createAdapter } from '@socket.io/redis-adapter';
import { createClient } from 'redis';

async function bootstrap() {
  const app = await NestFactory.create(AppModule);

  const pubClient = createClient({ url: process.env.REDIS_URL });
  const subClient = pubClient.duplicate();
  await Promise.all([pubClient.connect(), subClient.connect()]);

  const io = app.get(Server);  // get the socket.io server
  io.adapter(createAdapter(pubClient, subClient));

  await app.listen(3000);
}
```

With the Redis adapter, emitting to a room from any server instance reaches all clients in that room, regardless of which instance they're connected to.
