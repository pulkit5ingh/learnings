## Goal
Model all four relation types in TypeORM, understand eager vs lazy loading trade-offs, and write complex queries using QueryBuilder.

## Background

TypeORM supports all relational database relationship types via decorators. Each relation decorator takes a type function (to avoid circular import issues) and a relation options object.

**ManyToOne / OneToMany** is the most common — a Post belongs to one User (ManyToOne on Post side), a User has many Posts (OneToMany on User side). The foreign key column lives on the ManyToOne side (Post table).

**ManyToMany** requires a join table. TypeORM can create it automatically using `@JoinTable()` on one side. Example: Articles and Tags.

**OneToOne** has the foreign key on the side that uses `@JoinColumn()`. Example: User and UserProfile.

**Cascade** options (`cascade: true` or `cascade: ['insert', 'update']`) tell TypeORM to automatically persist related entities when the parent is saved. Use carefully — it can cause unexpected bulk operations.

**Eager loading** (`eager: true`) automatically loads the relation whenever the entity is queried. **Lazy loading** returns a `Promise<T>` that you must await. The recommended approach is explicit loading using `relations` option or QueryBuilder's `.leftJoinAndSelect()` — it gives you control over what gets loaded per query.

**QueryBuilder** is TypeORM's fluent API for complex SQL: joins, subqueries, window functions, CTEs.

### 1. Entities

**`src/users/entities/user.entity.ts`** (extend from A2.1):
```typescript
import { Entity, PrimaryGeneratedColumn, Column, OneToMany, OneToOne } from 'typeorm';
import { Post } from '../../posts/entities/post.entity';
import { UserProfile } from './user-profile.entity';

@Entity('users')
export class User {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  name: string;

  @Column({ unique: true })
  email: string;

  // OneToOne with UserProfile
  @OneToOne(() => UserProfile, profile => profile.user, { cascade: true })
  profile: UserProfile;

  // OneToMany with Post
  @OneToMany(() => Post, post => post.author, { cascade: ['insert'] })
  posts: Post[];
}
```

```typescript
import { Entity, PrimaryGeneratedColumn, Column, OneToOne, JoinColumn } from 'typeorm';
import { User } from './user.entity';

@Entity('user_profiles')
export class UserProfile {
  @PrimaryGeneratedColumn()
  id: number;

  @Column({ nullable: true })
  bio: string;

  @Column({ nullable: true })
  avatarUrl: string;

  @OneToOne(() => User, user => user.profile)
  @JoinColumn()   // foreign key lives here: user_id column on user_profiles table
  user: User;
}
```

**`src/posts/entities/post.entity.ts`**
```typescript
import {
  Entity, PrimaryGeneratedColumn, Column,
  ManyToOne, ManyToMany, JoinTable, CreateDateColumn,
} from 'typeorm';
import { User } from '../../users/entities/user.entity';
import { Tag } from './tag.entity';

@Entity('posts')
export class Post {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  title: string;

  @Column('text')
  content: string;

  @Column({ default: false })
  published: boolean;

  // ManyToOne — foreign key: author_id on posts table
  @ManyToOne(() => User, user => user.posts, { onDelete: 'CASCADE' })
  author: User;

  // ManyToMany — join table: posts_tags
  @ManyToMany(() => Tag, tag => tag.posts, { cascade: true })
  @JoinTable()
  tags: Tag[];

  @CreateDateColumn()
  createdAt: Date;
}
```

```typescript
@Entity('tags')
export class Tag {

  @Column({ unique: true })
  name: string;

  @ManyToMany(() => Post, post => post.tags)
```

### 2. PostsService with explicit relation loading

**`src/posts/posts.service.ts`**
```typescript
import { Injectable, NotFoundException } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { Post } from './entities/post.entity';
import { Tag } from './entities/tag.entity';

@Injectable()
export class PostsService {
  constructor(
    @InjectRepository(Post)
    private readonly postRepository: Repository<Post>,
    @InjectRepository(Tag)
    private readonly tagRepository: Repository<Tag>,
  ) {}

  async create(userId: number, title: string, content: string, tagNames: string[]): Promise<Post> {
    // find-or-create tags
    const tags = await Promise.all(
      tagNames.map(async name => {
        let tag = await this.tagRepository.findOne({ where: { name } });
        if (!tag) {
          tag = this.tagRepository.create({ name });
          await this.tagRepository.save(tag);
        }
        return tag;
      }),
    );

    const post = this.postRepository.create({
      title,
      content,
      author: { id: userId },
      tags,
    });
    return this.postRepository.save(post);
  }

  // Explicit relation loading via find options
  async findAll(): Promise<Post[]> {
    return this.postRepository.find({
      relations: { author: true, tags: true },
      order: { createdAt: 'DESC' },
    });
  }

  // QueryBuilder for complex queries
  async findPublishedByTag(tagName: string): Promise<Post[]> {
    return this.postRepository
      .createQueryBuilder('post')
      .leftJoinAndSelect('post.author', 'author')
      .leftJoinAndSelect('post.tags', 'tag')
      .where('post.published = :published', { published: true })
      .andWhere('tag.name = :tagName', { tagName })
      .orderBy('post.createdAt', 'DESC')
      .getMany();
  }

  // Aggregate with QueryBuilder
  async getPostCountByUser(): Promise<{ userId: number; count: string }[]> {
    return this.postRepository
      .createQueryBuilder('post')
      .select('post.authorId', 'userId')
      .addSelect('COUNT(post.id)', 'count')
      .groupBy('post.authorId')
      .getRawMany();
  }

  async findOneWithAll(id: number): Promise<Post> {
    const post = await this.postRepository.findOne({
      where: { id },
      relations: { author: { profile: true }, tags: true },
    });
    if (!post) throw new NotFoundException(`Post ${id} not found`);
    return post;
  }
}
```

### 3. OneToOne cascade example

```typescript
// Creating a user with profile in one save()
const user = userRepository.create({
  name: 'Bob',
  email: 'bob@example.com',
  profile: {
    bio: 'Backend developer',
    avatarUrl: 'https://example.com/bob.jpg',
  },
});
// cascade: true on the User.profile relation means profile is inserted automatically
await userRepository.save(user);
```

### 4. ManyToMany manipulation

```typescript
// Add a tag to an existing post
const post = await postRepository.findOne({
  where: { id: 1 },
  relations: { tags: true },
});
const newTag = await tagRepository.findOne({ where: { name: 'nestjs' } });
post.tags.push(newTag);
await postRepository.save(post);

// Remove a tag
post.tags = post.tags.filter(t => t.name !== 'nestjs');
await postRepository.save(post);
```

### 5. Pagination with QueryBuilder

```typescript
async paginate(page: number, limit: number): Promise<{ data: Post[]; total: number }> {
  const [data, total] = await this.postRepository.findAndCount({
    relations: { author: true },
    order: { createdAt: 'DESC' },
    skip: (page - 1) * limit,
    take: limit,
  });
  return { data, total };
}
```
